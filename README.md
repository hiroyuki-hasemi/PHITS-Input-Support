# PHITS in Visula Studio Code

PHITSインプットファイル作成用Visual Studio Code Extension

## 機能

### シンタックスハイライト

- セクションタイトル
- パラメータ
- コメント
- 定数
- 数式
- 特定のセクション等で用いるキーワード
    - m, mat
    - set
    - infl
    - tr
